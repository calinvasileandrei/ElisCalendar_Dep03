To use the software you need to create a google developer outh token.
First go to google developer console api and create a new project
Go into the section credentials and download the json file for the credentials
Rename the file into => credentials.json
Add the file in the credentials folder

Start the script, the first time a new chrome window will pop up and you need to log in with the same google account
you used to create the credentials file.

Allow all the permission for the calendar

Close the window

The program should work now!
